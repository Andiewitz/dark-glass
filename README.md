# Dy's Portfolio

A personal portfolio site built with Next.js 16, React 19, Tailwind CSS v4, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19, Tailwind CSS v4
- **Animation:** Framer Motion (motion)
- **Language:** TypeScript
- **Fonts:** Plus Jakarta Sans, DM Sans, JetBrains Mono

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
src/
├── app/            # Next.js app router (layout, page, global styles)
├── components/
│   ├── layout/     # Navbar, Footer, Container, SectionWrapper
│   ├── sections/   # Section components + registry
│   └── ui/         # Reusable UI primitives (Button, Reveal)
├── config/         # Site configuration
├── content/        # Section content data
└── lib/            # Types, utilities
```
